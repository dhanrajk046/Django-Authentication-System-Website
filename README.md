# userproject

## Django Authentication System

A secure **Django-based Authentication System** that manages user access through login protection. Users must authenticate themselves before accessing protected pages; otherwise, they are automatically redirected to the login page.

This project demonstrates the implementation of **Django’s built-in authentication system**, including user login, logout, session management, and route protection.

### Features

* User Login and Logout functionality
* Authentication using Django's built-in auth system
* Protected routes (only accessible after login)
* Automatic redirection to the login page for unauthorized users
* Clean and simple UI for authentication pages

### Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** SQLite

### Purpose

This project is built to demonstrate **secure authentication and access control in Django web applications**, which is an essential feature for real-world web development.
